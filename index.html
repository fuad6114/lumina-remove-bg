API Reference

GET /health

Check that the API and model are online.

curl http://localhost:8000/health # → {"status":"ok","model":"u2net"}
POST /remove-background

Returns the processed PNG binary directly. Best for direct image consumption.

curl -X POST \ http://localhost:8000/remove-background \ -F "file=@photo.jpg" \ --output result.png
POST /remove-background/json

Returns base64-encoded PNG + metadata. Ideal for JavaScript frontends.

fetch('/remove-background/json', { method: 'POST', body: formData // FormData with 'file' field }) .then(r => r.json()) .then(({ image_base64, width, height, processing_time_s }) => { img.src = `data:image/png;base64, ${image_base64}`; });
